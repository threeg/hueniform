# Frontend

React + Vite + TypeScript SPA. React Router, TanStack Query, CSS Modules.

- **Wireframes are binding:** `docs/04-wireframes/` defines screens, states and navigation.
  Implement to the wireframes; raise and change the doc first if a wireframe is wrong.
- **API contract:** `docs/03-api-contract.md` defines the HTTP shapes. The frontend is
  built against the contract independently of the backend (CONVENTIONS.md §4.5).
- **Test targets:** `make test-frontend` for component tests; `make test-e2e` for
  user-flow tickets.
- **Tests:** `frontend/src/test/`
